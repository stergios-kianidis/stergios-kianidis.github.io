Καλησπέρα αγαπητέ υποψήφιε συνεργάτη.


σου γράφουμε για να σε ενημερώσουμε ότι μελετήσαμε το βιογραφικό σου για την αγγελία μας ως Web Developer και σε έχουμε επιλέξει, ανάμεσα σε άλλους υποψηφίους, για το επόμενο βήμα της διαδικασίας. Το βήμα αυτό περιλαμβάνει ένα challenge που έχουμε ετοιμάσει, μέσα από το οποίο ευελπιστούμε να αξιολογήσουμε το επίπεδο γνώσης των υποψηφίων σε html, css, javascript και php. Έχοντας διαβάσει τα ζητούμενα σε αυτό το mail, θα μπορείς να εκτιμήσεις αν η υλοποίησή του είναι εντός των τεχνικών σου γνώσεων και αν αποφασίσεις να το προχωρήσεις, δε θα πρέπει να σου πάρει πάνω από 2.5 ώρες. Οι υποψήφιοι των οποίων τα παραδοτέα κριθούν ικανοποιητικά θα προχωρήσουν σε ραντεβού στα γραφεία μας.

Περιγραφή του challenge


Στα συνημμένα αρχεία, παρουσιάζεται ένα dummy mockup (μία responsive σελίδα) σε desktop και mobile version, μαζί με τις εικόνες που το απαρτίζουν. Στόχος είναι να υλοποιήσεις την απαραίτητη html, css, javascript και php ώστε το αποτέλεσμα να είναι όσο πιο κοντά στο mockup, βάσει των screenshots και του guideline. Για την υλοποίηση μπορείς να χρησιμοποιήσεις όποιο τρόπο θεωρείς πιο κατάλληλο, δεν υπάρχει περιορισμός (πχ grid system). Σαν τελικό παραδοτέο, περιμένουμε ένα folder με τα απαραίτητα αρχεία php (ή/και html), js, css και εικόνες.  Το τελικό παραδοτέο, θα πρέπει να παράγει validated HTML από W3C και μπορεί να αναπαραχθεί σωστά σε όλους τους σύγχρονους browsers (IE >11).


Γενικές οδηγίες


H γραμματοσειρά είναι roboto bold (include από google fonts).

Για font-sizes και gutters / margins χρησιμοποίησε ότι σου κάθεται σωστά στο μάτι.

Στα screenshots φαίνεται το στήσιμο σε desktop και mobile. Για tablet περιμένουμε την πρότασή σου.

Τα blocks 2-7 είναι links και στο hover θέλουμε το άσπρο background να γίνει μαύρο και οι τίτλοι άσπροι, με ένα transition 300ms. Το click (αντίστοιχα με το mouse-over) πρέπει να πραγματοποιείται σε ολόκληρο το block.

Το περιεχόμενο του κάθε block (τίτλος, εικόνα και url) θα πρέπει να φορτώνεται δυναμικά και τυχαία σε κάθε load, μέσα από σχετικό php αρχείο που θα δημιουργήσεις.


Laptop / Desktop


Στο desktop_grid.jpg θα δεις τη σελίδα με ένα grid και με αρίθμηση στα διάφορα blocks (για reference). Στο desktop.jpg φαίνεται η σελίδα όπως πρέπει να υλοποιηθεί. Το laptop_aboveFold.jpg δείχνει πώς θα φαίνεται το site στο desktop/laptop version.


Το block 1 πρέπει να πιάνει το 100% του φάρδους και του ύψους της εκάστοτε οθόνης (fullscreen). Μέσα στο block κεντράρεται το svg logo καθέτως και οριζοντίως.

Το block 1 περιέχει 2 τίτλους και 2 urls. Στόχος είναι το hover και το click να πιάνονται σε όλο το block και να πηγαίνουν στο url του μεγάλου τίτλου. Κλικάροντας συγκεκριμένα το δεύτερο τίτλο να σε πηγαίνει στο url αυτού. Το 2ο url μπορεί να είναι hard-coded πάνω στο site (δεν είναι ανάγκη να αλλάζει σε κάθε reload).

Τα block 3&4 καταλαμβάνουν αθροιστικά (μαζί με το κενό μεταξύ τους) το ύψος του block 1 (μεταβαλλόμενο αναλόγως του περιεχομένου του)

Τα blocks 5,6&7 να είναι ίσα σε ύψος, βάσει του ύψους που καταλαμβάνει το block με το περισσότερο content.


Mobile


Στο mobile.jpg θα δεις πώς προσαρμόζεται η σελίδα σε κινητές συσκευές.

Το block 1 καταλαμβάνει το 100% του φάρδους και το 50% του ύψους της οθόνης. To svg logo κεντράρεται μέσα στο block οριζοντίως και καθέτως

Τα υπόλοιπα blocks παρουσιάζονται όλα με τον ίδιο τρόπο, το καθένα καταλαμβάνοντας το ύψος που απαιτείται από το content του.
